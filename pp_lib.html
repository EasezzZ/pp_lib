This is the documentation for the routines in the pp_lib library.

Installation

The whole library is provided as a tar file (below), which contains the source code (in pp_lib/src) and documentation (in pp_lib/doc). There are no external dependencies To install the library, simply unpack the tar file and add the directory pp_lib/pro to your IDL path.
For help on the many ways to add the directory to the IDL path, you can see IDL's help, or the relevant part in David Fanning's instructions on how to set the path for his libraries.
